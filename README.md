# first-logic-test

나의 첫 번째 논리 테스트 프로젝트

## 설명
이 저장소는 조건문(conditional logic)과 관련된 간단한 예제 및 연습용 노트북을 저장한 곳입니다. 주로 Colab에서 실행하도록 만든 노트북을 포함합니다.

## 주요 파일
- `conditional_logic_test.ipynb` — 조건문 실습 노트북 (Colab에서 실행)  
  https://colab.research.google.com/github/ltaegyu2010/first-logic-test/blob/main/conditional_logic_test.ipynb

## 사용 방법 (Colab)
1. 위 Colab 링크를 클릭하거나, GitHub에서 노트북 파일을 열고 "Open in Colab" 버튼으로 실행합니다.  
2. Colab에서 셀을 순서대로 실행하세요.

로컬에서 실행(스크립트인 경우)
1. 필요한 패키지 설치:
```
pip install -r requirements.txt
```
2. 스크립트 실행:
```
python scripts/conditional_logic_test.py
```

## 간단한 코드 예시
```python
x = 10
if x > 5:
    print("x는 5보다 큽니다")
else:
    print("x는 5 이하입니다")
```
